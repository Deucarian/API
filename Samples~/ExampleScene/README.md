# API Example Scene

Open `APIExample.unity`, assign the included client and endpoint assets to the
`API Example` GameObject, and enter Play Mode. The controller demonstrates
authentication, GET and POST requests, cancellation, and `ApiResult` handling
without introducing application-specific services.

The example targets JSONPlaceholder by default. Review the endpoint URLs before
running it in a network-restricted environment, and leave `runOnStart` disabled
when you want to trigger requests from your own test controls.
