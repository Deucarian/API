# Support for Deucarian API

Use [GitHub Issues](https://github.com/Deucarian/API/issues) for reproducible API package defects, documentation gaps, and narrowly scoped feature requests. Search existing issues before opening a new one.

Include the package version, Unity editor version, target platform, request/response shape with credentials and personal data removed, the smallest reproduction you can provide, and the full error or stack trace. A minimal project or test is especially helpful for serialization, transport, cancellation, and AssetBundle problems.

`main` is the stable package channel. `develop` is a preview channel and may change before promotion. Please reproduce against the applicable current channel before reporting an older snapshot.

Do not disclose vulnerabilities, tokens, private endpoints, or user data in an issue. Follow [SECURITY.md](SECURITY.md) for private security reports.

Support is provided on a best-effort basis. General Unity account, licensing, platform, and editor support remains with Unity.
